# ALTO OCTANAJE

##  Método de marketing estratégico - Pasantía UNIR

Propuesta Comercial (venta)
    ↓
F1_Protocolo_Onboarding    → sesión 1: diagnóstico
    ↓
Checklist + Dashboard      → cálculo del IMM
    ↓
Informe de Diagnóstico     → firma → F2 abierta
    ↓
Buyer Persona + Estrategia de Marca  → co-construcción
    ↓
OKRs + Plan de Marketing   → firma → F3 abierta
    ↓
F3F4_Protocolo_Entrega     → sesión F3.1: canales y presupuesto
                           → sesión F3.2: materiales comerciales
                           → sesión F3.3: validación + test autonomía
    ↓
                           → sesión F4.1: Dashboard configurado
                           → sesión F4.2: cierre + acta firmada
                           → revisión trimestral agendada
