# ALTO OCTANAJE

##  Método de marketing estratégico - Pasantía UNIR

1) Propuesta Comercial (venta)
    ↓
2) F1_Protocolo_Onboarding    → sesión 1: diagnóstico

3)  Checklist + Dashboard      → cálculo del IMM

5)  Informe de Diagnóstico     → firma → F2 abierta

7)  Buyer Persona + Estrategia de Marca  → co-construcción
   
9)  OKRs + Plan de Marketing   → firma → F3 abierta
    
11)   3F4_Protocolo_Entrega     → sesión F3.1: canales y presupuesto
                           → sesión F3.2: materiales comerciales
                           → sesión F3.3: validación + test autonomía
                           → sesión F4.1: Dashboard configurado
                           → sesión F4.2: cierre + acta firmada
                           → revisión trimestral agendada
